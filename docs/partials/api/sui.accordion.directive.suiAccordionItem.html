suiAccordionItem
directive in module sui.accordion

Description

The accordion item of suiAccordion. It requires the controller of its parent suiAccordion. So it cannot be used without suiAccordion.

Demo

Usage

as element:
<sui-accordion-item
       title="{string}"
       active="{boolean}">
</sui-accordion-item>
as attribute
<ANY sui-accordion-item
     title="{string}"
     active="{boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
titlestring

Title

Be active or not
