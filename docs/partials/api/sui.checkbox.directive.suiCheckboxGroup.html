suiCheckboxGroup
directive in module sui.checkbox

Description

A checkbox group. Check suiCheckbox.

Usage

as element:
<sui-checkbox-group
       model="{array}"
       disabled
       name="{string}"
       options="{array}"
       on-check="{function}">
</sui-checkbox-group>
as attribute
<ANY sui-checkbox-group
     model="{array}"
     disabled
     name="{string}"
     options="{array}"
     on-check="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
modelarray

The values of selected items.

disabledboolean

Disabled or not

namestring

Name for grouping

optionsarray

A list of options to populate the checkboxes. Every option should contain label and value.

on-checkfunction

The function to be called when the checkbox is toggled.
