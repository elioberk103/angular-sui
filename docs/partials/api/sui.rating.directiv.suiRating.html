suiRating
directive in module sui.rating

Description

Rating directive

Usage

as element:
<sui-rating
       size="{enum}"
       ui-style="{enum}"
       options="{array}"
       disabled
       on-leave="{function}"
       on-hover="{function}"
       on-rate="{function}">
</sui-rating>
as attribute
<ANY sui-rating
     size="{enum}"
     ui-style="{enum}"
     options="{array}"
     disabled
     on-leave="{function}"
     on-hover="{function}"
     on-rate="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
sizeenum

"mini" || "tiny" || "small" || "large" || "huge" || "massive" || empty

ui-styleenum

"star" || "heart"

optionsarray

Array of integer values, e.g., [1, 2, 3, 5, 8, 13]

disabledboolean

Disabled or not

on-leavefunction

Callback when leaving the rating

on-hoverfunction

Callback when hovering the rating

on-ratefunction

Callback when confirming the rating
