suiSelect
directive in module sui.select

Description

Form select field.

Usage

as element:
<sui-select
       model="{string}"
       options="{object}"
       indicating-text="{string}"
       disabled
       label="{string}"
       searchable="{boolean}"
       ajax-url="{string}">
</sui-select>
as attribute
<ANY sui-select
     model="{string}"
     options="{object}"
     indicating-text="{string}"
     disabled
     label="{string}"
     searchable="{boolean}"
     ajax-url="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
modelstring

Value of the selected option

optionsobject

Options of the select

indicating-textstring

Text when no default option is selected

disabledboolean

The select is disabled or not

labelstring

The text shown above the select

searchableboolean

Whether searchable or not

ajax-urlstring

If specified, go to fetch options from this URL
