suiCheckbox
directive in module sui.checkbox

Description

Checkbox element

Usage

as element:
<sui-checkbox
       label="{string}"
       model="{boolean}"
       disabled
       ui-style="{enum}"
       on-check="{function}">
</sui-checkbox>
as attribute
<ANY sui-checkbox
     label="{string}"
     model="{boolean}"
     disabled
     ui-style="{enum}"
     on-check="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
labelstring

Label

modelboolean

The model of this directive

disabledboolean

Disabled or not

ui-styleenum

"checkbox" || "toggle" || "slider"

on-checkfunction

The function to be called when the checkbox is toggled.
