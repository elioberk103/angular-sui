suiAccordion
directive in module sui.accordion

Description

Accordion to switch to different sub-sections. It has ng-transclude: true to include an array of directive suiAccordionItem.

Usage

as element:
<sui-accordion
       on-switch="{function}">
</sui-accordion>
as attribute
<any sui-accordion
     on-switch="{function}">
   ...
</any>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
on-switchfunction

Callback when switching between items
