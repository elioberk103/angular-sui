suiRadioGroup
directive in module sui.radio

Description

A form radio group.

Usage

as element:
<sui-radio-group
       model="{string}"
       disabled
       options="{array}"
       name="{string}"
       inline="{boolean}"
       on-check="{function}">
</sui-radio-group>
as attribute
<ANY sui-radio-group
     model="{string}"
     disabled
     options="{array}"
     name="{string}"
     inline="{boolean}"
     on-check="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
modelstring

The value of selected item

disabledboolean

Whether disabled or not

optionsarray

A list of options to populate the radios. Every option should contain label and value.

namestring

Name of the radio group

inlineboolean

If true, the radio items are put as inline elements

onCheckfunction

Callback function when checked.
