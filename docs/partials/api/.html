Description

A single radio. It is used by suiRadio.

Demo

Usage

as element:
<
       model<i="{string}"
       disabled
       label="{string}"
       value="{string}"
       on-check="{function}">
</>
as attribute
<ANY 
     model<i="{string}"
     disabled
     label="{string}"
     value="{string}"
     on-check="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
modelstring

class="exchange icon"> The value

disabledboolean

Whether disabled or not

labelstring

The label text displayed next to the radio

valuestring

Value for grouping

onCheckfunction

Callback function when checked.
