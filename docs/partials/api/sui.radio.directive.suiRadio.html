suiRadio
directive in module sui.radio

Description

A single radio. It is used by suiRadioGroup.

Demo

Usage

as element:
<sui-radio
       model<i="{string}"
       disabled
       label="{string}"
       value="{string}"
       on-check="{function}">
</sui-radio>
as attribute
<ANY sui-radio
     model<i="{string}"
     disabled
     label="{string}"
     value="{string}"
     on-check="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
modelstring

class="exchange icon"> The value

disabledboolean

Whether disabled or not

labelstring

The label text displayed next to the radio

valuestring

Value for grouping

on-checkfunction

Callback function when checked.
