suiProgress
directive in module sui.progress

Description

A progress bar.

Usage

as element:
<sui-progress
       percentage="{int}"
       size="{enum}"
       color="{string}"
       disabled
       show-percentage="{boolean}"
       bottom-attached="{boolean}"
       top-attached="{boolean}"
       type="{enum}"
       active="{boolean}"
       indicating="{boolean}"
       label="{string}">
</sui-progress>
as attribute
<ANY sui-progress
     percentage="{int}"
     size="{enum}"
     color="{string}"
     disabled
     show-percentage="{boolean}"
     bottom-attached="{boolean}"
     top-attached="{boolean}"
     type="{enum}"
     active="{boolean}"
     indicating="{boolean}"
     label="{string}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
percentageint

[0, 100] integer number indicating percentage

sizeenum

"mini" || "tiny" || "small" || "large" || "huge" || "massive" || empty

colorstring

Color of the bar

disabledboolean

Disabled or not

show-percentageboolean

Whether to show the percentage number on the bar

bottom-attachedboolean

Attach the bar to bottom of the transcluded content

top-attachedboolean

Attach the bar to top of the transcluded content

typeenum

"success" || "warning" || "error"

activeboolean

Whether show animation on the bar

indicatingboolean

If true: different color when the value is different. Colors are: gray->red->yellow->green

labelstring

If not empty, display the label under the bar
