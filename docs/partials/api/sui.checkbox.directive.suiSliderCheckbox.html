suiSliderCheckbox
directive in module sui.checkbox

Description

Checkbox element with ui-style as slider.

It is an alias of suiCheckbox with a specific UI style. Check suiCheckbox.

Usage

as element:
<sui-slider-checkbox
       label="{string}"
       model="{boolean}"
       disabled
       on-check="{function}">
</sui-slider-checkbox>
as attribute
<ANY sui-slider-checkbox
     label="{string}"
     model="{boolean}"
     disabled
     on-check="{function}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
labelstring

Label

modelboolean

The model of this directive

disabledboolean

Disabled or not

on-checkfunction

The function to be called when the checkbox is toggled.
