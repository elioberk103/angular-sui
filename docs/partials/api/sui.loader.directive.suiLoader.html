suiLoader
directive in module sui.loader

Description

A dimmed loader background when you are loading something.

Usage

as element:
<sui-loader
       loading-text="{string}"
       active="{boolean}"
       size="{enum}"
       dim-whole-page="{boolean}">
</sui-loader>
as attribute
<ANY sui-loader
     loading-text="{string}"
     active="{boolean}"
     size="{enum}"
     dim-whole-page="{boolean}">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
loading-textstring

The text to be displayed under the spinner icon

activeboolean

To display dimmer or not

sizeenum

"mini" || "tiny" || "small" || "large" || "huge" || "massive" || empty

dim-whole-pageboolean

To dim the whole page or not
