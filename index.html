Angular-sui

Angular directives for Semantic-UI.

Download this project as a .zip file Download this project as a tar.gz file

angular-sui is a collection of AngularJS directives for semantic-ui.

Interactions supported by jQuery are reimplemented by Angular.

Note: angular-sui is not ready for production.
